SurviveJS - Webpack and React

Go from zero to Webpack and React hero

What will I learn?

Facebook's React powers an increasing amount of sites. There is a good reason for this. It simplifies frontend development greatly. Combined with Webpack, a bundling tool, you have a powerful frontend development environment in your hands.

SurviveJS - Webpack and React shows you how to build a simple Kanban application based on these technologies. During the process you will learn at least the following things:

  1. To set up a Webpack based development and production environment. You will learn to get most out of react-hot-loader for development.
  2. To improve the quality of your code by using ESLint and spot possible mistakes earlier.
  3. To use Babel to unlock the future features of JavaScript now.
  4. To understand the basics of React and get into the mindset. Even though React is conceptually simple, it has to "click" in order to become powerful.
  5. To learn the basics of Alt, an implementation of Flux Application Architecture. Flux complements React well.
  6. To move beyond TODO application level and into application design and architecture.
  7. To style your React application in various emerging ways.
  8. To implement drag and drop for your application using React DnD
  9. To deploy your application into production environment using various deployment strategies.
  10. To author developer friendly JavaScript libraries for other developers to consume.

If even one of those points sounded interesting, please check out the online version of the book. The book source is available at GitHub.
